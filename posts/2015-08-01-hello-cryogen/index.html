Hello Cryogen

I've been meaning to switch to a statically-generated blog for a while, and specifically to Cryogen hosted on GitHub Pages, for a combination of several reason:

  • I am already writing my posts in Markdown for Ghost, and prior to that, Scriptogr.am
  • Overhead of maintaining a Ghost blog is hosted on OpenShift Online
  • I prefer Clojure to JavaScript
  • Cryogen developer lacarmen is amazingly responsive. This request for tag-specific RSS feeds is done within a day. Ghost dragged its feet on this.

Will set up Disqus integration later – and need to work on a better-looking site theme too, but that can wait. For now, hello!
